# AI Guardian

## Description
AI Guardian is a web application developed using Python and Flask. It provides an AI-powered assistant through a simple and user-friendly interface.

## Features
- AI-based assistant
- Simple and responsive user interface
- Fast responses
- Easy to use

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## Project Structure
AI-GUARDIAN/
├── app.py
├── templates/
├── static/
├── requirements.txt
└── README.md

## How to Run

1. Install the required packages:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open your browser and visit:
   http://127.0.0.1:5000
